# mern-bug-tracker

Objective: Implement testing and debugging best practices in a MERN-style Bug Tracker application.

## Project structure
- `backend/` — Express backend with model/service/controller structure and tests.
- `frontend/` — React frontend with components, error boundary and tests.

## Install & run backend
```bash
cd backend
npm install
npm start
# dev: npm run dev (requires nodemon)
